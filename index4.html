UL LI Generator PRO (FINAL)

Fungsi:
Mengubah daftar keyword menjadi banyak file HTML berisi link, lalu otomatis di-zip.

Format URL:
/categories/?q=baris 01
(Default jika domain kosong)

Aturan pembersihan data:
  • Spasi depan & belakang dihapus
  • Spasi berlebih dirapikan
  • Tanda _ dan - diubah jadi spasi
  • Simbol di awal/akhir dihapus
Filter baris:
  • Baris kosong → dibuang
  • Hanya simbol → dibuang
  • Kurang dari 5 huruf/angka → dibuang
Mode split:
  • Bagi jumlah bagian → menentukan jumlah file
  • Item per bagian → menentukan isi tiap file
Contoh:
10.000 baris + 1000 item → hasil 10 file dalam ZIP
